## [1.0.0] - 2026-03-29

### 🎉 初始版本发布

#### 新增功能
- **风险登记台账** - 风险识别、评估、等级计算、自动编号
- **风险矩阵** - 可视化风险评估矩阵图
- **风险应对措施** - 措施跟踪、预算管理、进度监控
- **KRI监控预警** - 关键指标监控、趋势分析、自动预警
- **应急预案管理** - 预案编制、演练记录、版本控制
- **风险检查表** - 检查任务、问题整改、验收跟踪
- **供应商风险管理** - 供应商评估、智能评分、风险评级
- **培训管理** - 培训记录、效果评估、统计报表
- **系统管理** - 部门管理、用户管理、角色权限

#### 技术特性
- Next.js 14 + React + TypeScript
- Prisma ORM + PostgreSQL
- JWT 认证 + Cookie 管理
- Tailwind CSS + Radix UI
- 多租户架构
